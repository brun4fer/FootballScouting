# FootballScouting

Plataforma de scouting para observacao, comparacao e evolucao de jogadores.

## Stack

- Next.js App Router
- TypeScript
- Drizzle ORM
- Neon DB
- Tailwind CSS
- Recharts

## Rotas principais

- `/configuracoes/epocas`
- `/configuracoes/competicoes`
- `/configuracoes/posicoes`
- `/configuracoes/clubes`
- `/configuracoes/paises`
- `/jogadores`
- `/relatorios`
- `/consulta/*`
- `/equipa-sombra/*`
- `/estatisticas`

## Scripts

- `npm run dev`
- `npm run build`
- `npm run db:generate`
- `npm run db:migrate`
- `npm run db:push`
- `npm run db:studio`
- `npm run db:seed`
